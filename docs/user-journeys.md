# User Journeys V1

## User Types

### 1. Sinh viên tìm phòng

Mục tiêu:

- Tìm phòng phù hợp với ngân sách.
- Xem thông tin chi tiết.
- Liên hệ Sale.

Journey:

Truy cập website
→ Tìm phòng
→ Lọc theo khu vực
→ Lọc theo giá
→ Xem chi tiết phòng
→ Bấm liên hệ Zalo
→ Sale hỗ trợ

---

### 2. Sinh viên tìm người ở ghép

Mục tiêu:

- Tìm người phù hợp để ở chung.

Journey:

Truy cập website
→ Vào mục ở ghép
→ Lọc theo khu vực
→ Lọc theo giới tính
→ Xem chi tiết bài đăng
→ Bấm liên hệ Zalo
→ Sale hỗ trợ

---

### 3. Khách hàng cần đăng nhu cầu ở ghép

Mục tiêu:

- Tìm thêm người ở chung phòng.

Journey:

Liên hệ Sale
→ Cung cấp thông tin
→ Sale ghi nhận
→ Admin tạo bài đăng
→ Bài đăng xuất hiện trên website
→ Có người liên hệ
→ Sale hỗ trợ ghép

---

### 4. Sale

Mục tiêu:

- Tiếp nhận lead.
- Tư vấn khách hàng.
- Chốt phòng.

Journey:

Khách liên hệ Zalo
→ Tiếp nhận thông tin
→ Tư vấn
→ Hẹn xem phòng
→ Chốt phòng

---

### 5. Admin

Mục tiêu:

- Quản lý dữ liệu hệ thống.

Journey:

Đăng nhập
→ Quản lý phòng
→ Quản lý bài ở ghép
→ Theo dõi lead
→ Cập nhật dữ liệu
